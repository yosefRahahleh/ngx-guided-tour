ngx-guided-tour Demo

Guided tour is a great way to introduce your users to new features or remind them how to use exisiting features. Hover your mouse over the orb above to start the tour.

Orbs are optional, If not set or set to false the tour will just start.




Click restart to start the tour again.

There are multiple features to correctly position and caclulate scrolling on your tours.
